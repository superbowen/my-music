# my-music

> 我的第二个vue项目：音乐播放器demo
> 项目地址 `https://github.com/superbowen/my-music`

## 技术栈
> * MVVM框架： Vue 2.2
> * 源码：es6
> * 代码风格检查：eslint
> * 构建工具：webpack
> * 前端路由：vue-router
> * 状态管理：vuex
> * 服务端通讯：vue-resource

## Build Setup

``` bash
# 安装
npm install

# 运行
npm run dev

# 生产
npm run build

```
